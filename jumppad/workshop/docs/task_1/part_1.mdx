# Part 1: Deploy your Minecraft server with Terraform

## Machine Name

To access the minecraft server you can use the following DNS name in your minecraft client.

```
{{ machine_url }}
```

<Task id="apply_terraform_configuration">
To deploy the Minecraft server you need to run the following commands:
```
cd terraform
terraform init
terraform apply
```
</Task>



Did you manage that? If not you can find the solution below.

<Solution title="Solution: configure kubernetes auth">

```bash
vault write auth/kubernetes/config \
  token_reviewer_jwt=@/usr/src/k8s.token \
  kubernetes_host="https://kubernetes.default.svc:443" \
  kubernetes_ca_cert=@/usr/src/k8s.crt
```

</Solution>
