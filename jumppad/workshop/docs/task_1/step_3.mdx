# Step 3: Deploy Your Minecraft Server
1. Use SSH to connect to your Minecraft server and verify it is running:

```
ssh packer@localhost -p2222 
``` 
2. Select Yes for known host then input `packer` as password when prompted

3. To access the minecraft server you can use the following DNS name in your minecraft client.

```
{{ machine_url }}
```
