## Login to HCP Vault

1. Navigate to the Instrqut terminal tab. 
2. Set environment variable
```
export VAULT_ADDDR=https://skoaap-public-vault-7a278d60.8c03478c.z1.hashicorp.cloud:8200/
```
3. Using the Vault CLI, login using
```
vault login -namespace=admin -method=userpass username=<team_name> password='<password>'
```
