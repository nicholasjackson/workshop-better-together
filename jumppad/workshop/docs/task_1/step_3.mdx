# 🏴‍☠️Step 3: Connect to your Minecraft Server

1. Go to your Minecraft client launcher and click the **Play** button.
![Minecraft Launcher](/assets/task_1a/mc_launcher.png)

2. Choose **Multiplayer** option as shown below.
![Multiplayer](/assets/task_1a/mc_player.png)

3. Click the **Add Server**.
![Add Server](/assets/task_1a/server_addr.png)

4. Give a custom name such your team name in the **Server Name** field and copy the following DNS name at the **Server Address** field. Hit **Done**. 
```
{{ machine_url }}
```
5. If everything is running correctly, you will see a 5 green bar symbol next to 0/100.
![Server Added](/assets/task_1a/server_added.png)
