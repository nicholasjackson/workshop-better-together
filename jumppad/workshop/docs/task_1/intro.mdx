# Introduction

## In this task you are going to deploy and new Minecraft server in your environment using Terraform.

* Part 1: Create a new Terraform configuration that configures the Libvirt provider.
* Part 2: Create a new Virtual machine using the `libvirt_domain` resource for the provider.
* Part 3: Connect to the minecraft server and verify that it is running.
* Part 4: Complete the task in Minecraft to find Blackbeards treasure.
