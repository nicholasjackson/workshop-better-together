# Step 3: Update machine credentials in AAP
In this step, you will update the machine credentials in Automation Automation Platform (AAP) to use Hashicorp Vault Signed SSH credential plugin configured in the previous step.
Machine credentials are associated with Ansible Job templates and allow for authenicated access to a target machine. 
The machine credentials are used to authenticate with the target machine and execute the Ansible playbook.
<Task id="configure_machine_creds">
1. From the left menu, click on **Automation Execution** and then **Credentials**. The `minecraft_ssh_credentials` has been preconfigured but you'll need to input your ssh key for Hashicorp Vault key signing
![machinecreds](https://dl.dropbox.com/scl/fi/341ju2vyxuyewtmtugifd/machine_creds.png?rlkey=fi1yf5c2aa1op1zpu1uo6byf8&st=f9v01n50&dl=0)
2.Edit the machine credentials.
![editcreds](https://dl.dropbox.com/scl/fi/5xhymq46zgmmg7fokddz6/edit_creds.png?rlkey=t7stlggwekmqspmkqqd39m43c&st=277hqdtg&dl=0)
3. Update the credentials with the private SSH key from Hashicorp Vault.
![updatecreds](https://dl.dropbox.com/scl/fi/inxuavg90revxwvs2zlnt/update_creds.png?rlkey=agh5s6hq9lqfbelg32wgegvjn&st=do3xae7n&dl=0)
4. Configure and test signed SSH key from Hashicorp Vault
![testcreds](https://dl.dropbox.com/scl/fi/imql8636prbjh266ajko3/test_creds.png?rlkey=jirr6jockxoil2uwgcsn1n7v1&st=673mumkf&dl=0)
5. If the test is successful, click finish and **Save credential** to save the configuration.
![savecreds](https://dl.dropbox.com/scl/fi/6afakdgv8ogitvnicf3m7/save_creds.png?rlkey=uj3mjb8418y6nyp9mid50zqp4&st=y31vunx2&dl=0)
</Task>
