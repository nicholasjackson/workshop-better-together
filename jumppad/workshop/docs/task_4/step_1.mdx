# Step 1: Gaining the Access 
Enable access to the shared Minecraft server by creating a new AAP inventory and executing the existing minecraft AAP job template.
This job template will enable access to the server by adding your username to the server configuration using Ansible.

* The debug output from the job execution displays the results of updating the Minecraft server's access list, confirming the changes were applied
