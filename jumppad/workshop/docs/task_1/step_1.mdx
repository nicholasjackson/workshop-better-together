# Step 1: Initialize the Environment:
 * Open the Instruqt terminal and navigate to the Terraform project directory.
 * Run the following command to initialize Terraform:

 ```
cd terraform
terraform init
```
* This downloads the necessary Libvirt provider plugins required to create the virtual machine.

* Apply the Terraform configuration to deploy the Minecraft server:

```
terraform apply -auto-approve
```
* Terraform will provision a virtual machine that acts as your pirate world.
* Once complete, Terraform will output the serverâ€™s IP address.
